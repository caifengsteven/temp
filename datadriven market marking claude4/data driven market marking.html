A Data-Driven Deep Learning Approach for Options Market Making

Authors: Qianhui Lai, Xuefeng Gao, Lingfei Li

Journal: Quantitative Finance, 2023

1. Executive Summary

Key Innovation

This paper develops a data-driven approach for options market making that:

  • Discovers significant self-excitation in options market orders using empirical analysis
  • Models order flow with Hawkes processes with power-law kernels (non-Markovian)
  • Solves the market making problem using deep neural networks
  • Demonstrates superior out-of-sample performance compared to traditional approaches

1.1 Main Contributions

  1. Empirical Discovery: Market orders for options exhibit strong self-excitation but weak cross-excitation
  2. Model Selection: Hawkes process with time-varying baseline and power-law kernel fits the data best
  3. Methodology: Deep learning algorithm that handles non-Markovian order flows and constraints
  4. Performance: Using realistic models and Hawkes features significantly improves trading performance

2. Empirical Analysis of Options Market Orders

2.1 Data Description

Dataset Characteristics

  • Source: Chicago Board Options Exchange (CBOE)
  • Stocks: Apple (AAPL) - highly liquid, Bank of America (BAC) - medium liquid
  • Period: Full year 2019 (252 trading days)
  • Frequency: Tick-by-tick trades and 1-minute snapshots
  • Trade Classification: Lee-Ready algorithm (80% accuracy for CBOE data)

2.2 Self-Excitation Analysis

Nonparametric Hawkes Model

The intensity of order type \(i \in \{0:\text{buy}, 1:\text{sell}\}\) is modeled as:

$$\lambda^i(t) = \mu^i + \sum_{j \in \{0,1\}} \sum_{t_v^j < t} \phi^{j,i}(t - t_v^j)$$

where:

  • \(\mu^i\): Baseline intensity (constant in nonparametric approach)
  • \(\phi^{j,i}(\cdot)\): Kernel measuring influence of type \(j\) events on type \(i\)
  • \(t_v^j\): Occurrence time of \(v\)-th event of type \(j\)

Empirical Finding #1

Strong self-excitation: \(\phi^{0,0}\) and \(\phi^{1,1}\) show significant positive values

Weak cross-excitation: \(\phi^{0,1}\) and \(\phi^{1,0}\) are negligible compared to self-excitation

2.3 Parametric Model Estimation

Parametric Hawkes Process

Based on empirical findings, the intensity is modeled as:

$$\lambda(t) = \mu(t) + \sum_{t_v < t} g(t - t_v)$$

where the baseline intensity \(\mu(t)\) is:

  • AAPL: Piecewise linear on 6 intervals (U-shaped pattern)
  • BAC: Piecewise constant on 3 intervals (L-shaped pattern)

Two kernel specifications tested:

Power Law Kernel (Non-Markovian)
$$g(t - t_v) = \frac{\alpha}{(t - t_v + \delta)^{\beta}}$$
Exponential Kernel (Markovian)
$$g(t - t_v) = \alpha e^{-\beta(t - t_v)}$$

3. Market Making Problem Formulation

3.1 Setup and Dynamics

Market Making Framework

The market maker operates over discrete time grid \(\{t_0 = 0, t_1, ..., t_K = T\}\) and at each \(t_k\):

  • Posts bid order at distance \(\delta_k^b\) from mid-price
  • Posts ask order at distance \(\delta_k^a\) from mid-price
  • Bid price: \(p_k^b = C_k - \delta_k^b\)
  • Ask price: \(p_k^a = C_k + \delta_k^a\)

where \(C_k\) is the option mid-price at time \(t_k\).

Inventory Dynamics

Option inventory evolution:

$$q_k^o = q_k^o + Q_k^b(\delta_k^b) - Q_k^a(\delta_k^a)$$

Delta hedging (stock inventory):

$$q_k^s = -q_k^o \times \Delta_k$$

Cash dynamics:

$$x_{k+1} - x_k = (C_k + \delta_k^a) \times Q_k^a(\delta_k^a) - (C_k - \delta_k^b) \times Q_k^b(\delta_k^b) - \delta q_{k+1}^s S_{k+1}$$

3.2 Optimization Objective

Expected Utility Maximization

The market maker solves:

$$\max_{\delta_k^a, \delta_k^b > 0} \mathbb{E}[U(W_K - W_0)]$$

where the exponential utility function is:

$$U(y) = \begin{cases} \frac{1 - e^{-\gamma y}}{\gamma} & \text{if } \gamma > 0 \text{ (risk-averse)} \\ y & \text{if } \gamma = 0 \text{ (risk-neutral)} \end{cases}$$

Terminal wealth:

$$W_K = x_K + q_K^o C_K + q_K^s S_K$$

3.3 Market Impact Model

Execution Probability

The probability that a limit order at distance \(\delta\) is executed given \(N_{mkt}\) market orders:

$$\Pr(Q(\delta) = 1 | N_{mkt}) = 1 - (1 - e^{-\kappa \delta})^{N_{mkt}}$$

where \(\kappa > 0\) is the market impact parameter.

Intuition: Each market order has probability \(e^{-\kappa \delta}\) of reaching the limit order.

4. Deep Learning Solution Approach

4.1 Neural Network Architecture

Network Structure

  • Architecture: Feedforward Neural Network (FNN)
  • Hidden layers: 2 layers with 64 neurons each
  • Activation (hidden): ReLU(\(x\)) = max(\(x\), 0)
  • Activation (output): Sigmoid(\(x\)) = \(\frac{1}{1 + e^{-x}}\)
  • Total networks: 2K (one for bid, one for ask at each time)

4.2 Feature Engineering

Input Features for Neural Networks

State variables (always included):

  • \(S_k\): Stock price
  • \(C_k\): Option price
  • \(x_k\): Cash position
  • \(q_k^o\): Option inventory
  • \(q_k^s\): Stock inventory

Additional Hawkes features:

  1. Intensity feature: $$f_{b,k} = \lambda^0(t_k) = \mu^0(t_k) + \sum_{t_v^0 < t_k} \frac{\alpha^0}{(t_k - t_v^0 + \delta^0)^{\beta^0}}$$
  2. Expected arrivals feature: $$f_{b,k} = \mathbb{E}[N_{k,k+1}^0 | \mathcal{F}_k^0]$$

    Computed via Volterra integral equation:

    $$\psi_k^i(u) = 1 + \int_0^u h_k^i(u-s)\psi_k^i(s)ds$$

4.3 Training Algorithm

Deep Learning Training Procedure

  1. Data Generation:
    • Simulate N paths of stock prices (Black-Scholes model)
    • Calculate option prices and deltas
    • Simulate market order arrivals (Hawkes process)
  2. Forward Pass:
    • For each \(t_k\), compute \(\delta_k^a\), \(\delta_k^b\) from neural networks
    • Simulate order executions based on market arrivals
    • Update inventories and cash
  3. Loss Function: $$\mathcal{L}(\theta) = -\frac{1}{N} \sum_{j=1}^N U(W_K^j(\theta) - W_0)$$
  4. Optimization: Adam optimizer with minibatch SGD

5. Empirical Results

5.1 Experimental Setup

Testing Methodology

  • Training: Month n data → Generate strategy

  • Rolling window: 11 months of out-of-sample testing
  • Trading frequency: 1 minute (AAPL), 5 minutes (BAC)
  • Initial conditions: \(x_0 = \$5000\), \(q_0^o = 10\) contracts

6. Extension to Multiple Options with Constraints

6.1 Constrained Optimization Problem

Multi-Option Market Making with Greeks Constraints

$$\begin{align} \max_{\delta_{k,l}^a, \delta_{k,l}^b} &\quad \mathbb{E}[U(W_K - W_0)] \\ \text{s.t.} &\quad |\Gamma_K| \leq M_\Gamma \\ &\quad |\mathcal{V}_K| \leq M_\mathcal{V} \\ &\quad |q_{K,l}^o| \leq M_{Q,l} \quad \forall l = 1,...,m \end{align}$$

where \(\Gamma_K\) and \(\mathcal{V}_K\) are portfolio gamma and vega at terminal time.

6.2 Penalty Method Solution

Unconstrained Formulation

Convert to unconstrained problem using quadratic penalties:

$$\min_{\delta} -\mathbb{E}\left[U(W_K - W_0) - \eta_1 P_\gamma - \eta_2 P_\nu - \eta_3 \sum_{l=1}^m P_{q,l}\right]$$

where penalty function: \(P = \max\{0, x-y\}^2 + \min\{0, x+y\}^2\) for constraint \(|x| \leq y\)

7. Key Insights and Implications

Why This Approach Works

  1. Data-Driven Modeling: Discovering actual market microstructure patterns rather than imposing assumptions
  2. Non-Markovian Dynamics: Power-law kernels capture long memory in order flow
  3. Feature Engineering: Hawkes intensity and expected arrivals provide valuable predictive information
  4. Flexible Framework: Deep learning handles complex dynamics and constraints naturally

7.1 Practical Implications

For Market Makers

  • Model Selection: Use Hawkes processes, not Poisson, for options order flow
  • Feature Design: Include order flow predictions in decision-making
  • Time-of-Day Effects: Adapt quotes to intraday patterns (U-shape, L-shape)
  • Risk Management: Implement Greeks constraints to limit overnight risk

7.2 Limitations and Extensions

Current Limitations

  • Simple Black-Scholes model for underlying (could use stochastic volatility)
  • No joint modeling of option prices and order flow
  • Fixed order size (could optimize quantity as well)
  • No consideration of trading fees/rebates

Future Research Directions

  • Multi-asset market making with correlation
  • Adversarial settings with strategic traders
  • Reinforcement learning comparisons
  • High-frequency implementation (sub-second)

8. Conclusion

Main Takeaways

  1. Empirical Foundation: Options market orders exhibit self-excitation best captured by Hawkes processes with power-law kernels
  2. Methodological Contribution: Deep learning provides effective solution for non-Markovian market making problems
  3. Performance Gains: Using realistic models and appropriate features significantly improves out-of-sample performance
  4. Practical Framework: Approach handles real-world constraints (Greeks, inventory) naturally

This data-driven approach demonstrates the importance of understanding actual market microstructure
rather than imposing convenient but unrealistic assumptions in quantitative trading strategies.
